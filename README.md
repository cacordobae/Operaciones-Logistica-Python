# Operaciones y Logística — Modelado con Python

Modelado de procesos operativos: inventario, lead time, cumplimiento 
de pedidos y optimización de rutas. Aplica simulación y análisis 
estadístico a problemas de cadena de suministro.

## Qué resuelve
Un jefe de operaciones necesita identificar cuellos de botella, 
medir el fill rate y simular escenarios de demanda.

## Datos
Dataset sintético de pedidos, inventario y proveedores.

## Qué incluye
- Cálculo de fill rate y lead time
- Simulación Montecarlo de demanda
- Identificación de cuellos de botella
- Visualizaciones

## Cómo verlo
Notebook Jupyter o Google Colab.
